## Project Description
    This website hosts NASA's Earth Polychromatic Camera (EPIC) and other NASA photos. In addition, this website contains an EPIC image collection of the Earth's lighted surface. The Polychromatic Earth website displays the latitude and longitude of the Earth's surface; clicking each latitude's view button reveals the Earth's sides.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

The `pages/api` directory is mapped to `/api/*`. Files in this directory are treated as [API routes](https://nextjs.org/docs/api-routes/introduction) instead of React pages.
